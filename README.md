# bn-api-node
Bigneon Javascript Middleware

# Install

`git clone https://github.com/big-neon/bn-api-node.git`

`npm install`

# Running

`npm run test`

`npm run build`

`npm run build-web`

`npm run build-server`

`npm run build-tsc`

`npm run db-seed`


# Development
For local development using bn-web, link bn-node-api as an npm module in bn-web:

`cd bn-api-node/`

`npm link`

`cd ../bn-web`

`npm link bn-api-node`
